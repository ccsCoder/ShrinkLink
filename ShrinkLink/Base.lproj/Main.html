Shrink My Link

Steps to enable extension for MacOS
  1. Launch Safari
  2. Click on Safari -> Settings in the menubar, or Prese CMD + , ( Command + Comma )
  3. Select the "Extensions" tab, and location the "ShrinkLink Extension"
  4. Select the "Extensions" tab, and location the "ShrinkLink Extension" in the left pane
  5. Click the Checkbox to enable the extension.
Steps to enable extension for iPhone / iPad
  1. Launch Safari
  2. Click on AA in the address bar
  3. Select "Manage Extensions" and Enable the ShrinkLink extension.
  4. To use, click AA on the address bar, and select ShrinkLink extension from the menu.

The extension will request for accessing your current tab in order to read the URL you want to shorten. Additionally it might request for Cliboard Permissions to enable Copying the shortened URL. Enjoy !
